Plugins/CSS Editor
==================


[Plugins](../Plugins.1.html "Plugins")

About CSS Editor 
-------------------------------------------------------------------------

The CSS Editor plugin allows users to make changes to the CSS (code
which controls appearance) of their Omeka installation.

This plugin will be most helpful to people with experience using CSS and
designing sites.

Requirements
-----------------------------------------------------------------

CSS Editor requires Omeka 2.0 or higher.

Installation
-----------------------------------------------------------------

You can download the latest version of CSS Editor from the [Omeka Plugin Directory](../../plugins.html). To add the plugin to your Omeka site,
follow the installation instructions on the [Managing Plugins 2.0](../Managing_Plugins_2.0.html#Installing_a_Plugin "Managing Plugins 2.0")
page.

Configuration
-------------------------------------------------------------------

Access the plugin from the Plugins menu in the top navigation bar of the
Omeka Admin. Click the blue Configure button to go to the CSS Editor.

The CSS Editor configuration page contains a large text field in which
you can write your custom css. Type your code as you would in a css
document. Be sure to save changes.
