Plugins/RecordRelations 2.0
===========================

<div id="contentSub">

<span class="subpages">&lt;
[Plugins](../Plugins.1.html "Plugins")</span>

</div>

<div id="primary">

Record Relations is a dependency for some plugins, but provides no
functionality for the end user of its own. You only need Record
Relations if another plugin requires it.

Once you have installed Record Relations, you can activate and
deactivate it. It has no settings.

<span id="For_Developers" class="mw-headline">For Developers</span>
-------------------------------------------------------------------

Record Relations is an abstraction of the Item Relations plugin,
allowing relationships between any record type in Omeka. Plugins that
use Record Relations should build their own interfaces for users to
build relations between records.
