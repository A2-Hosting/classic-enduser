Plugins/TagBandit
=================

<div id="contentSub">

<span class="subpages">&lt;
[Plugins](../Plugins.1.html "Plugins")</span>

</div>

<div id="primary">

[TagBandit](../../plugins.html#tagbandit) allows users to conduct a
search of items in the administrative interface and then assign tags to
a selected group of items.

<span id="Installation" class="mw-headline"> Installation </span>
-----------------------------------------------------------------

1.  Upload the Timeline plugin directory to your server. See See:
    [Installing a Plugin](../Installing_a_Plugin.html).
2.  Activate the plugin from the admin → Settings → Plugins page.
