Release Notes for 1.5.1
=======================

<div id="primary">

Omeka 1.5.1 is the first maintenance release for the 1.5 series. It was
released on April 11, 2012.

<span id="Bug_Fixes" class="mw-headline"> Bug Fixes </span>
-----------------------------------------------------------

-   [\#161](https://github.com/omeka/Omeka/issues/161): omeka-xml output
    only renders the last element of an element set *(reported
    by annewootton)*
-   [\#164](https://github.com/omeka/Omeka/issues/164): Media helper
    uses nonexistent \$path when a dimension is given *(reported
    by symac)*
-   [\#165](https://github.com/omeka/Omeka/issues/165): Element dropdown
    doesn't get translated *(reported by mjlassila)*
