<span id="Installation" class="mw-headline"> Installation </span>
-----------------------------------------------------------------

1\. Upload and install the Sitenotes plugin into your plugins folder on
the server, see: ([Installing a Plugin](../Installing_a_Plugin.html)).

2\. Activate this from the Plugins page in your admin panel:
/admin=&gt;Settings=&gt;Plugins.

<span id="Using_Sitenotes" class="mw-headline"> Using Sitenotes </span>
-----------------------------------------------------------------------

1\. Once activated, a new tab will appear on the top navigation bar of
the admin panel. To write notes for all users, edit the panel and add
instructions, style sheets, interpretations of Dublin Core fields, or
any other notes for the back-end users.
