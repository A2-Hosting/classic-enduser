Functions/element exists
========================

<div id="contentSub">

<span class="subpages">&lt;
[Functions](../Functions.html "Functions")</span>

</div>

<div id="primary">

Checks to see if a given element exists, and is part of a specified
element set.

<span id="Usage" class="mw-headline"> Usage </span>
---------------------------------------------------

<div class="mw-geshi mw-content-ltr" dir="ltr">

<div class="php source-php">

``` {.de1}
<?php element_exists($elementSetName, $elementName) ?>
```

</div>

</div>

<span id="Arguments" class="mw-headline"> Arguments </span>
-----------------------------------------------------------

-   \$elementSetName (*string*): The element set to check.
-   \$elementName (*string*) The name of the element to check

<span id="Example" class="mw-headline"> Example </span>
-------------------------------------------------------

This function can generally be used to check if an element exists before
displaying it

<div class="mw-geshi mw-content-ltr" dir="ltr">

<div class="php source-php">

``` {.de1}
<?php 
if ( element_exists('Dublin Core', 'Date Added') ) {
    echo item('Dublin Core', 'Date Added');
}
?>
```

</div>

</div>

---------------------------------------------------------------------------------
